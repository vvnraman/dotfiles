Git configuration#

The git setup consists of 3 parts

  1. The OS specifc config contains the base structure.

    This exists primary for windows and linux so that we can look for the presence of tools in $PATH in the OS native locations trivially.

  2. User specific config to specify basic git info

  3. Optional tool integrations if they are available

Load order#

  1. symlink_config.tmpl - Chezmoi symlink template which resolves to an os specific config.

    symlink_config.tmpl#
    1{{- if eq .chezmoi.os "windows" -}}
2windows.gitconfig
3{{- else -}}
4linux.gitconfig
5{{- end -}}

    Linux and Windows base behavior is defined as follows:

    linux-config.tmpl#
     1{{- /* vim: set ft=gotmpl: */ -}}
 2{{- if and (eq .chezmoi.os "linux") (eq .chezmoi.osRelease.id "arch") -}}
 3linux-arch
 4{{- else if and (eq .chezmoi.os "linux") (eq .chezmoi.osRelease.id "ubuntu") -}}
 5linux-ubuntu
 6{{- else if and (eq .chezmoi.os "linux") (eq .chezmoi.osRelease.id "redhat") -}}
 7linux-redhat
 8{{- else -}}
 9linux
10{{- end -}}
    windows.gitconfig.tmpl#
    1# vim: ft=gitconfig
2[include]
3    path = ~/.config/git/user-{{ template "user-config.tmpl" . }}.gitconfig
4{{ if findExecutable "delta" (list "C:\\Users\\vvnra\\AppData\\Local\\Microsoft\\WinGet\\Links") }}
5    path = ~/.config/git/tool-delta.gitconfig
6{{ end }}
7{{ if findExecutable "kdiff3" (list "C:\\Users\\vvnra\\AppData\\Local\\Programs\\KDiff3\\bin") }}
8    path = ~/.config/git/tool-kdiff3.gitconfig
9{{ end }}

  2. user-vvnraman.gitconfig - gets included by the os config

    The os specific config resolve the user config file using user-config.tmpl template.

    user-config.tmpl#
    1{{- /* vim: set ft=gotmpl: */ -}}
2{{- last (splitList "\\" .chezmoi.username) -}}

  3. Tool configs if they’re present.

    • tool-delta.gitconfig

    • tool-kdiff3.gitconfig
